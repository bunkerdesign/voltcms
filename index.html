setup

Volt Core Bric has to be placed inside the global footer area in a bloc at the bottom of the page. It will bring the basic configuration and needed files into the project and after export on your server.

Create a new bloc with padding=none and place it at the lowest point in the document tree inside the global footer area. Add the Volt Core bric inside this new bloc, without any other content inside that bloc. Having other content in that bloc might lead to wrongly generated files.

volt cms brics

volt blog

Volt Blog can be used for many different things, like a blog for writing news articles, a FAQ page, or a product catalog.
In order to install a Volt Blog, search for the Volt Blog Bric and add it to the page. You can have multiple blogs per project, and one blog per page.

Add the Volt Blog Editor bloc (not bric) to the page. This is a modal UI only visible on the published server when logged in.
When both Volt Blog bric and Volt Blog Editor bloc are placed on the page, you will get a new (add) button when logged in for adding a new blog post. Selecting this add button will open the editor modal UI.

More info

volt content

Add this bric in order to determine the editable area. After that you will be able to add and remove content directly from your prefered browser. Both mobile and desktop. Also you can add as many brics as you want.
Easy as that.

More info

volt gallery

This bric is really similar to Volt Content bric but directly dedicated to gallery functionality.
Add this bric and it will allow you to create and moderate your gallery.

More info

volt protect

In order to password protect a page, you have to put the Volt Protect bric on the page, which should only be able to be accessed by logged in users. If someone wants to access the protected page, and he’s not logged in, he will be redirected to the login page. After successful login, he will be redirected back to the protect page.

More info

volt recent posts

In order to display recent posts from one of the blogs to a page, the Volt Recent Posts bric has to be added to the page where these should appear. This can be the blog page itself, or another page, like the home page displaying posts from a blog page.
